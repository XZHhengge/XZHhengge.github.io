**先进行连接，再互相传输数据,过程不能中断，云**

```python
ip:47.92.222.94，port:8888,type:tcp
```
**连接：**

```python
from表明是哪一个4g卡
{'action':'connect', 'from': 'A/B/C'}
响应："connect_success"
```
**发送**

```python
 to: 要发送到哪个卡，message：发送的内容
{'action':'send', 'to': 'A/B/C', 'message': '***'}
 响应："send_succes"
```
